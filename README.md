# Memory Game

## Overview
A simple game in vanilla JavaScript, HTML, and CSS.

The objective of the Memory Game, also known as Concentration and Matching Pairs, is to match pairs of cards by turning two over at a time.

## Running the Game
To play the game, click [here](https://kianezaz.github.io/memory-game/)

## Instructions
1. Select the desired number of cards to play with
2. Click on any two cards on the grid to view the NBA logo they contain
3. If the logos on the two selected cards match, they will be removed from the grid
4. Once every card is removed from the grid, you win the game!

**Note**: Try to complete the game in the least number of moves and least amount of time possible. These statistics will be displayed upon completion of the game.
